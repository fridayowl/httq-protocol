## [1.0.0] - 2025-12-30

### Added
- Initial release of HTTQ Protocol
- HTTQ-LATTICE-1024/2048/4096 algorithms
- Post-quantum key encapsulation mechanism
- Quantum-resistant digital signatures
- Hybrid encryption mode (classical + PQC)
- Express/Node.js middleware
- Browser client SDK
- Interactive web demo
- Comprehensive documentation
- Protocol specification (RFC-style)
- Migration guide from HTTPS
- Performance benchmarks
- Security audit guidelines

### Features
